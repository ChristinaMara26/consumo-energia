Este projeto foi desenvolvido para a disciplina de DS-I. O Objetivo é calcular o consumo mensal de um eletrodoméstico e o seu custo estimado em Reais. 
## O programa é desenvolvido em python
1. Solicita o nome do aparelho
2. Solicita a potência em Watts
3. Solicita o tempo médio de uso diario em horas
4. Calcula o consumo mensal (KWh) e o custo final; considerou o preço do KWh é igual a R$ 0,75/KWh
